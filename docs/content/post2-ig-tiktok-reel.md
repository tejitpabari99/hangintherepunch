## Music

Pick ONE of these approaches:
- **Best:** Open TikTok → search "emotional" or "wholesome" in sounds → pick whatever soft piano track is trending that week
- **Safe choice:** Search for "Einaudi" or "Gymnopédie" on CapCut's music library (royalty-free versions exist)
- **If using IG Reel editor:** Browse their music library for "emotional" — Instagram has licensed tracks built in

The music should be soft, building slightly toward the end. NOT sad throughout — it should feel hopeful by slides 9-10.

---

## CapCut Instructions (if you haven't used it)

1. Download CapCut (free, iOS/Android/Desktop)
2. New Project → import all photos in order
3. Set each photo to its duration (2-4 sec per the table above)
4. Add text overlays for each slide (white text, clean font like "Inter" or "Montserrat")
5. Add transitions between photos: "Dissolve" for most, "None/Cut" for slide 7 (the bullying one — the sudden cut creates emotional impact)
6. Add music track, lower volume to ~30-40% so it doesn't overpower
7. Export at 9:16 ratio (vertical/portrait) for Reels + TikTok
8. Export at 1:1 if you also want a square version
